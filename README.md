# 使用 `nuxt.js` 做的一个博客

- 可以用 `markdown` 形式进行博客记录
- nuxt 静态部署
- 不定时更新

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# generate static project
$ npm run generate

# preview 
$ npm run preview
```

For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
